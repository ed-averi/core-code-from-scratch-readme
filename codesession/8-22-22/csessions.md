# Typescript Introduction

## Create Folder

```shell
  $ mkdir <FOLDER_NAME>
```

## Go Into Folder

```shell
  $ cd <FOLDER_NAME>
```

## Get Out Of Folder

```shell
  $ cd ..
```

## Clean Terminal (Windows)

```shell
  $ cls
```

## Clean Terminal (Linux & Mac)

```shell
  $ clean
```

## Install Node

[Here](https://nodejs.org/en/) is the link to install node

## Check Node Version

```shell
  $ node --version
```

## Check npm Version

```shell
  $ npm --version
```

## Start npm Project

```shell
  $ npm init --y
```

## Install Typescript (local)

```shell
  $ npm install typescript
```

## Compile Typescript File

```shell
  $ npx tsc <FILE_NAME>.ts
```

## Install All Node Packages

```shell
  $ npm install
```

## Node RPL State

```shell
  $ node
```

## Run Javascript File On Node

```shell
  $ node <FILE_NAME>.js
```
